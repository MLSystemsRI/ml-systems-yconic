# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | Yes       |

## Reporting a Vulnerability

To report a security vulnerability, please email security@mlsystemsri.com.

Do not open a public GitHub issue for security vulnerabilities.

We will acknowledge receipt within 48 hours and provide a detailed response within 7 days.

## Known Considerations

- TTP verification fees are calculated server-side; never trust client-submitted scores
- RCM financial calculations are for modeling only; consult qualified financial professionals
- AI crawler detection uses user-agent matching and can be spoofed; enforce server-side
