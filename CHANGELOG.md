## [1.0.0] - 2026-03-29

### Added
- Transparency Trust Protocol (TTP) engine — score 0-100, 5 access bands, AI crawler detection, regulator scoping, verification fees
- Reverse Construction Mortgage (RCM) engine — 6 credit tiers, 100% principal allocation, standard + preferred schedules, overpayment modes
- Intent Schema — Lucent Lens scoring, MVE gate, Custodian constraints, agent intent factory, anti-SaaS pricing rules
- 135 unit tests across all engines (80% coverage thresholds)
- GitHub Actions CI pipeline (typecheck, lint, test, format)
- API documentation, architecture guide, contributing guide
- Demo landing pages and 28 Three.js visualizations
